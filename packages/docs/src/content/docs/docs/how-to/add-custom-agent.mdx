---
title: Add a Custom Agent
description: Declare a new agent with prompt, mode, model intent, skills, triggers, and policy.
---

Add an `agent` block to `.weave/config.weave`.

```text
agent database-reviewer {
  description "Reviews persistence and migration safety"
  prompt_file "database-reviewer.md"
  models ["claude-sonnet-4-5"]
  mode subagent
  temperature 0.1
  skills ["sql", "code-review"]

  tool_policy {
    read allow
    write deny
    execute deny
    network deny
    delegate deny
  }

  triggers [
    { domain "Database" trigger "Schema changes, migrations, query safety" }
  ]
}
```

Then create `.weave/prompts/database-reviewer.md`.

Validate and inspect:

```bash
bun packages/cli/src/main.ts validate --project
bun packages/cli/src/main.ts prompt inspect database-reviewer
```

If the agent should delegate to other subagents, set `delegate allow`; otherwise it will have no delegation targets.
