---
title: Initialize Config
description: Create global or project .weave configuration files.
---

## Create project config

```bash
bun packages/cli/src/main.ts init --scope local --yes
```

This writes `.weave/config.weave` and creates `.weave/prompts/`.

## Create global config

```bash
bun packages/cli/src/main.ts init --scope global --yes
```

This writes `~/.weave/config.weave` and creates `~/.weave/prompts/`.

## Choose a non-default directory

```bash
bun packages/cli/src/main.ts init --scope local --install-dir .weave --yes
```

For ordinary use, keep the canonical paths because config discovery checks `~/.weave/config.weave` and `<project>/.weave/config.weave`.

## Configure a detected harness

```bash
bun packages/cli/src/main.ts init --scope local --harness opencode --yes
```

Only supported harness installers are installed. OpenCode is the implemented adapter today.
