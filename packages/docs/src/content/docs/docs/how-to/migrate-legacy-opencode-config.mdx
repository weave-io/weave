---
title: Migrate Legacy OpenCode Config
description: Convert legacy weave-opencode.jsonc files to canonical .weave DSL.
---

## Local migration

```bash
bun packages/cli/src/main.ts init migrate --scope local
```

Source: `./.opencode/weave-opencode.jsonc`

Destination: `./.weave/config.weave`

## Global migration

```bash
bun packages/cli/src/main.ts init migrate --scope global
```

Source: `~/.config/opencode/weave-opencode.jsonc`

Destination: `~/.weave/config.weave`

## Run non-interactively

```bash
bun packages/cli/src/main.ts init migrate --scope local --yes
```

If the destination exists, use `--force` to allow overwrite. The migrator creates a `.bak` backup before replacing an existing destination.

## After migration

Validate the generated DSL:

```bash
bun packages/cli/src/main.ts validate --project
```

The migration preserves the legacy source file and reports skipped fields as warnings.
