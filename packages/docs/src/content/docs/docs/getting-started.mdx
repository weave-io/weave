<p class="lede">Weave runs on Bun. Install the workspace, build the packages, validate a config, and learn the main packages you will work with.</p>

## Requirements

<p>Weave runs on <strong style="color:var(--text)">Bun</strong>. Node.js is not a supported runtime for the project itself.</p>

## Install and build

<CodeBlock name="terminal" id="gs-install">{`git clone https://github.com/weave-io/weave.git
cd weave
bun install
bun run build`}</CodeBlock>

## Common workspace commands

<CodeBlock name="terminal" id="gs-commands">{`bun run typecheck
bun test
bun run validate-config
bun run docs:dev`}</CodeBlock>
