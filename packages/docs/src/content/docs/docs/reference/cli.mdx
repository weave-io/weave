<p class="lede">Common Weave CLI entry points for validating configs, scaffolding new projects, and migrating legacy OpenCode setups.</p>

## `weave validate`

<p>Validate a <code>.weave</code> config.</p>

<CodeBlock name="terminal" id="cli-validate">{`weave validate --project
weave validate --path .weave/config.weave
weave validate --path .weave/config.weave --json`}</CodeBlock>

## `weave init`

<p>Create a new config interactively or non-interactively.</p>

<CodeBlock name="terminal" id="cli-init">{`weave init
weave init --scope local --yes
weave init --scope global --yes`}</CodeBlock>

## `weave init migrate`

<p>Migrate a legacy OpenCode config into the current <code>.weave</code> DSL.</p>

<CodeBlock name="terminal" id="cli-migrate">{`weave init migrate --scope local
weave init migrate --scope global`}</CodeBlock>
