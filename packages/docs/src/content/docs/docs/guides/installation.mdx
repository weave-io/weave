<p class="lede">Clone the repository, install dependencies with Bun, build the workspace, and validate the project config.</p>

## Requirements

<ul>
  <li><a class="link" href="https://bun.sh">Bun</a></li>
  <li>Git</li>
</ul>

## Clone and install

<CodeBlock name="terminal" id="in-clone">{`git clone https://github.com/weave-io/weave.git
cd weave
bun install`}</CodeBlock>

## Build the workspace

<CodeBlock name="terminal" id="in-build">{`bun run build`}</CodeBlock>

## Validate the project config

<CodeBlock name="terminal" id="in-validate">{`bun run validate-config`}</CodeBlock>

## Useful local commands

<CodeBlock name="terminal" id="in-local">{`bun run typecheck
bun test
bun run docs:dev`}</CodeBlock>
