---
title: Inspect Prompts
description: List resolved agents and render composed prompts before launching a harness.
---

## List agents

```bash
bun packages/cli/src/main.ts prompt list
```

This includes builtin agents, user agents, and generated category shuttles, excluding disabled agents.

## List as JSON

```bash
bun packages/cli/src/main.ts prompt list --json
```

## Inspect one prompt

```bash
bun packages/cli/src/main.ts prompt inspect loom
```

## Inspect metadata and prompt as JSON

```bash
bun packages/cli/src/main.ts prompt inspect loom --json
```

The JSON includes the final prompt, resolved mode, models, effective tool policy, skills, and delegation target names.

## Troubleshoot missing prompt files

User `prompt_file` and `prompt_append_file` values resolve relative to the config scope's `prompts/` directory. For a project config, `prompt_file "reviewer.md"` resolves to `.weave/prompts/reviewer.md`.
